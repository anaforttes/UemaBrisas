Não adicionar a linha `Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>` nas mensagens de commit.

**Why:** preferência explícita do usuário — quer os commits limpos, sem coparticipação da IA.

**How to apply:** em todos os commits neste projeto, omitir completamente o bloco Co-Authored-By.
